个人博客搭建

hexo博客框架+git部署

安装软件

git

  • window：git
  • linux(Ubuntu):sudo apt-get install git-core
  • linux(CentOS):sudo yum install git-core

node.js

  • window:node.js
  • linux:

Hexo

  • 通用：

npm $ npm install -g hexo-cli

cnpm $ cnpm install -g hexo-cli

设置镜像

npm包管理器，但国内镜像源的速度很慢，所以要利用npm安装个cnpm（淘宝源），速度才能提上去。

npm install -g cnpm --registry=https://registry.npm.taobao.org

搭建博客

  • ctrl+R打开cmd输入runas/noprofile/user:Administrator cmd 输入密码切换管理员模式

  • md 文件名或者mkdir 文件名创建新文件夹（如果在搭建过程中出错，就干掉这个文件夹重来）

  • hexo init使用hexo生成一个初始博客

  • hexo s启动博客

  • 访问域名：localhost:4000 访问网页

  • hexo n "my new post" 创建一篇新博文

  • 编辑文本文件在命令行输入notepad（这里是编辑器名） my new post进行编辑，完成后保存退出。也可使用markdown软件进行编辑。语法通用

  • 使用cd ..\..返回上两级路径，回到\blog目录。

  • hexo cleam使用hexo清理一下数据库。

  • hexo g generate static files生成静态文件缩写。（相当于刷新）

  • hexo s 重新启动。

localhost:4000在本地使用，用于测试。

部署到github来公开使用

  • 登入github账号。

  • 创建一个新仓库

  • 在cmd内输入npm install --save hexo-deployer-git，进行安装hexo版的git插件

  • 配置_config.yml文件 notepad _config.yml打开文件

  • 输入hexo d部署到github，输入git账号密码，仓库ctrl+R刷新看看是否部署成功。

  • 直接访问创建的域名zhangxiao117.github.io.git查看是否能访问。
